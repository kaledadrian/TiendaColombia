#  TiendaColombia - Sistema Django

Sistema web desarrollado en Django para la gestión de una tienda.

---

##  Tecnologías usadas

- Python
- Django
- MySQL
- HTML, CSS, Bootstrap
- JavaScript

---

##  Funcionalidades

- Gestión de usuarios
- Gestión de gastos
- Categorías
- Reportes
- Dashboard con gráficos
