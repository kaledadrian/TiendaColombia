# Sistema De Inventario

## Descripción

TiendaColombia es una aplicación web desarrollada utilizando Django y MySQL, diseñada para gestionar de manera eficiente la información relacionada con productos, clientes e inventario de una tienda. Este proyecto fue realizado con fines académicos para aplicar conocimientos de programación, desarrollo web y bases de datos.

## Objetivo General

Diseñar e implementar un sistema de información web que facilite la administración de los procesos básicos de una tienda, permitiendo el registro, consulta, actualización y organización de la información de manera eficiente.

## Tecnologías Utilizadas

* Python
* Django 5
* MySQL
* HTML5
* CSS3
* Bootstrap

## Instalación y Configuración

Para ejecutar el proyecto es necesario contar con Python, Django y MySQL instalados previamente. Además, se deben instalar las dependencias especificadas en el archivo `requirements.txt`, configurar la conexión a la base de datos y ejecutar las migraciones correspondientes para la creación de las tablas requeridas por el sistema.

## Funcionalidades Principales

* Registro, consulta, actualización y eliminación de productos.
* Administración de la información de los clientes.
* Gestión y control del inventario disponible.
* Administración de usuarios y permisos de acceso.
* Visualización y actualización de registros almacenados en la base de datos.
* Acceso al panel de administración proporcionado por Django.
* Organización y almacenamiento seguro de la información comercial.

## Base de Datos

El sistema utiliza MySQL como gestor de bases de datos para garantizar el almacenamiento, organización y administración eficiente de la información.
